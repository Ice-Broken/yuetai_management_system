
# 悦泰管理系统
## 项目启动

**因为使用到 node-sass 的缘故，为了对应 node 版本，node 版本请使用 16**
**MySQL 版本建议使用 5，如果版本为 8 请将密码验证设置为 mysql_native_password**

推荐使用 yarn 作为包管理  
使用 **npm i yarn -g** 来安装 yarn

**如何启动项目**
0.引入 SQL 文件  
1.cd frontend  
3.yarn dev  
4.cd backend  
6.yarn dev
**if you find 监听端口:6789, it already start**
